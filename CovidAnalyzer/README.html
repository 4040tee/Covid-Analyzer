Introduction

This project is designed to provide a retrieval of COVID-19 infection data for countries one or more selected countries, computes different statistics on the data, and renders the result on a graphical interface. 

* For a full description of the module, visit the project page:
https://owl.uwo.ca/access/content/group/c9cc47fe-7c6c-49b0-9dd5-247a2977f123/Project%20Resources/CS2212A-Project-Description.pdf

Instructions

Start and Run

1. To start this project, run the login class and input a combinations recorded in the txt file
e.g username:user password:1234

a)Input the sample combination b)Press Submit!

Add Countries

2.
a) Type a country into the country selection text box
b) Press +
c) Repeat this process for as many countries as desired

Remove Countries

3.
a) If desired, type a country into the country selection text box
b) Press -
c) Repeat this process for as many countries as desired

Select Analysis Type

4.
a) Press the drop down menu containing the analysis types
b) Select an analysis type from the drop down menu

Calculate Results

5.
a) Press Recalculate
b) Observe results in output list and on map
